Convert rotation from unit quaternion representation to 3x3 matrix. 

The unit quaternion representation of rotations is a 4-\/dimensional unit vector q, where ! q and -\/q map to the same rotation (and choosing a random normalized ! 4-\/vector chooses a uniformly random rotation). This format is convenient for constructing and combining rotations but annoying for applying rotations to specific vectors. Use this function to convert to an appropriate format for applying the rotations (see Wikipedia for details).
